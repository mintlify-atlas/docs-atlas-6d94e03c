## Report Sections

The system is organized into six main reporting tabs:

### 1. General

High-level overview of school operations:
- Student distribution by plan (4, 8, 12, 16 classes)
- Class status breakdown (pie chart)
- Most popular specialties (bar chart)
- Classes by day of week

### 2. Students (Alumnos)

Detailed student analytics:
- Active vs inactive counts
- Top students by completed classes
- Individual attendance records
- Absence tracking (ACA/ASA)

### 3. Classes (Clases)

Class performance metrics:
- Total, completed, cancelled counts
- Status distribution
- Weekly distribution patterns
- Completion rates

### 4. Instructors

Instructor workload analysis:
- Classes per instructor
- Completion vs cancellation rates
- Efficiency percentages
- Comparative performance

### 5. Horses (Caballos)

Horse usage statistics:
- Classes per horse
- School vs private horse usage
- Horses without classes
- Availability status

### 6. Trial Classes (Clases de Prueba)

Trial class tracking:
- Total trial classes
- New people vs existing students
- Conversion rate (trial → active student)
- Trial classes by specialty

## Chart Types

Reports use various visualization types:

### Bar Charts

Used for:
- Comparing categories (plans, days, specialties)
- Showing workload distribution
- Displaying rankings

**Example**: Classes by day of week

### Pie Charts

Used for:
- Showing proportions (status distribution)
- Part-to-whole relationships
- Percentage breakdowns

**Example**: Class status distribution

### Horizontal Bar Charts

Used for:
- Long category names (horse names, instructor names)
- Specialty comparisons
- Easier label reading

**Example**: Horse usage

### Stacked Bar Charts

Used for:
- Multiple data series (completed vs cancelled)
- Cumulative totals

**Example**: Instructor performance

### Tables

Used for:
- Detailed data with multiple metrics
- Exportable information
- Precise numbers

**Example**: Student attendance table

## Exporting Data

Many reports include an **Export** button:

<Steps>
  <Step title="Navigate to Tab">
    Go to the report tab you want to export (Students, Instructors, Horses)
  </Step>
  <Step title="Click Export Button">
    Click the **"Exportar"** button with download icon
  </Step>
  <Step title="File Downloads">
    Excel file (.xlsx) downloads automatically
  </Step>
  <Step title="Open in Spreadsheet">
    Open in Excel, LibreOffice, or Google Sheets
  </Step>
</Steps>

### What Gets Exported

The Excel file includes:
- All data from the current table
- Professional formatting
- Column headers
- Calculated percentages and totals
- Date range in filename

### Export File Naming

```
Reporte_[Type]_YYYY-MM-DD.xlsx

Examples:
Reporte_Asistencia_2026-03-04.xlsx
Reporte_Instructores_2026-03-04.xlsx
Reporte_Caballos_2026-03-04.xlsx
```
