## Export Process

The export process is consistent across all report types:

<Steps>
  <Step title="Navigate to Report Tab">
    Go to Reports section and select:
    - **Alumnos** for student data
    - **Instructores** for instructor data
    - **Caballos** for horse data
  </Step>
  <Step title="Set Date Range">
    Configure the analysis period at the top:
    - Start date (Fecha Inicio)
    - End date (Fecha Fin)
  </Step>
  <Step title="Click Export Button">
    Click the **"Exportar"** button with download icon (📅↓)
  </Step>
  <Step title="File Downloads">
    Excel file (.xlsx) downloads automatically to your browser's download folder
  </Step>
  <Step title="Open File">
    Open in Microsoft Excel, LibreOffice Calc, Google Sheets, or compatible application
  </Step>
</Steps>

<Info>
  The export button only appears on tabs that support exporting (Students, Instructors, Horses).
</Info>

## File Naming Convention

All exported files follow a consistent naming pattern:

```
Reporte_[Type]_YYYY-MM-DD.xlsx

Examples:
Reporte_Asistencia_2026-03-04.xlsx    ← Student attendance
Reporte_Instructores_2026-03-04.xlsx  ← Instructor performance
Reporte_Caballos_2026-03-04.xlsx      ← Horse usage
```

**Components**:
- **Reporte**: Fixed prefix
- **Type**: Report category (Asistencia, Instructores, Caballos)
- **Date**: Export date (YYYY-MM-DD format)
- **Extension**: .xlsx (Excel 2007+ format)

<Note>
  The date in the filename is the **export date**, not the reporting period. The period is shown inside the file.
</Note>

## Student Attendance Export

Exported from the **Alumnos** tab.

### File Contents

<Tabs>
  <Tab title="Headers">
    **Column Headers**:
    - Alumno (Student name)
    - Total (Total classes)

</Tabs>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Download Not Starting">
    If file doesn't download:
    - Check browser popup blocker
    - Ensure sufficient disk space
    - Try different browser
    - Check download folder settings
  </Accordion>
  
  <Accordion title="File Won't Open">
    If Excel file won't open:
    - Verify file extension (.xlsx)
    - Try opening in different app
    - Re-download the file
    - Check if file is corrupted
  </Accordion>
  
  <Accordion title="Formatting Lost">
    If colors/formatting missing:
    - Open in Microsoft Excel (best support)
    - Update your spreadsheet software
    - Check file downloaded completely
  </Accordion>
  
  <Accordion title="Wrong Data in Export">
    If data doesn't match screen:
    - Verify date range before exporting
    - Check that filters weren't applied
    - Refresh page and try again
  </Accordion>
</AccordionGroup>
